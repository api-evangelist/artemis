# Artemis API (artemis)
APIs for NASA's Artemis program - the next generation of lunar exploration missions

**URL:** [Visit APIs.json URL](https://api.nasa.gov/artemis)

## Timestamps

- **Created:** 2024-01-15 
- **Modified:** 2024-01-15 

## APIs

### Artemis Mission Data API
Access data about Artemis missions including mission plans, schedules, and status updates

**Human URL:** [https://www.nasa.gov/artemis](https://www.nasa.gov/artemis)


#### Tags:

 - space, moon, exploration, artemis, missions

#### Properties

- [Documentation](https://api.nasa.gov/artemis/docs)
- [OpenAPI](https://api.nasa.gov/artemis/v1/openapi.json)
- [Authentication](https://api.nasa.gov/artemis/auth)

### Artemis Spacecraft Telemetry API
Real-time and historical telemetry data from Artemis spacecraft including Orion capsule and SLS rocket

**Human URL:** [https://www.nasa.gov/artemis/spacecraft](https://www.nasa.gov/artemis/spacecraft)


#### Tags:

 - telemetry, spacecraft, orion, sls, real-time

#### Properties

- [Documentation](https://api.nasa.gov/artemis/docs/telemetry)
- [OpenAPI](https://api.nasa.gov/artemis/v1/telemetry/openapi.json)
- [WebSocket](wss://api.nasa.gov/artemis/v1/telemetry/stream)

### Artemis Lunar Gateway API
Information about the Lunar Gateway space station including modules, experiments, and docking schedules

**Human URL:** [https://www.nasa.gov/gateway](https://www.nasa.gov/gateway)


#### Tags:

 - gateway, space-station, lunar-orbit, international

#### Properties

- [Documentation](https://api.nasa.gov/artemis/docs/gateway)
- [OpenAPI](https://api.nasa.gov/artemis/v1/gateway/openapi.json)

### Artemis Astronaut API
Information about Artemis astronauts, crew assignments, training schedules, and biographies

**Human URL:** [https://www.nasa.gov/artemis/astronauts](https://www.nasa.gov/artemis/astronauts)


#### Tags:

 - astronauts, crew, training, personnel

#### Properties

- [Documentation](https://api.nasa.gov/artemis/docs/astronauts)
- [OpenAPI](https://api.nasa.gov/artemis/v1/astronauts/openapi.json)
